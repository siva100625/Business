Project Description

The project aims to develop a system that detects alcohol levels in sweat using a transdermal alcohol sensor. This system employs an ESP8266 Wi-Fi module programmed in Embedded C to wirelessly transmit data to a Django backend server, which processes and stores the information. A React Native application retrieves this data and displays real-time alcohol readings. If the detected alcohol level exceeds a predefined threshold, an LED light blinks as a visual alert, and an email notification is sent to the owner, ensuring prompt awareness of any potential risks. This integrated solution promotes safety by providing immediate feedback on alcohol consumption levels.
